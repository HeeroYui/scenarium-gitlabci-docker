scenarium-gitlabci-docker
=========================

docker image for scenarium gitlab-ci

Build the image:
================

Execute the command:


```
docker image build -t scenarium-gitlabci .
```

the build is done  when you see:

```
Successfully tagged scenarium-gitlabci:latest
```
